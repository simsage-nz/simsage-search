== SimSage Search Plugin ==
Contributors: Rock de Vocht
Tags: plugin, commercial, readme
Author URI: https://simsage.ai
Requires at least 5.0
Stable Tag: 1.0.0
Description: this Plugin adds SimSage Semantic Search and bots and operator help to your web-site.
             you will need to have ZipArchive enabled (e.g. https://uk.godaddy.com/community/GoDaddy-Web-Hosting/ZipArchive-need-to-be-enable/td-p/7942)
             on Ubuntu 18.04 run: sudo apt-get install php7.x-zip  (where 7.x is your PHP version, e.g. 7.2)
